# OSSfolio

> Your open-source identity, beyond GitHub.

OSSfolio gives every open-source contributor a shareable public profile at **ossfolio.me/username** — contributor-centric, not repo-centric.

## What It Shows

Merged PRs · Issues opened · Orgs contributed to · Contribution heatmap & streak · Tech stack · GSoC/GSSoC badges · Contributor score · Leaderboard

## Stack

| Layer | Tech |
|-------|------|
| Frontend | Next.js · TypeScript · Tailwind · shadcn/ui · Framer Motion |
| Backend | Supabase · PostgreSQL |
| API | GitHub GraphQL API |
| Deploy | Vercel |

## Get Started

```bash
git clone https://github.com/<your-org>/ossfolio.git
cd ossfolio
npm install
cp .env.example .env.local
npm run dev
```

See [CONTRIBUTING.md](CONTRIBUTING.md) for the full setup guide and how to make your first contribution.
